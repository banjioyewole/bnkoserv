AUTONOMOUS FLIGHT
Autonomous Drone Control with Computer Vision

For my Capstone in Computer Science, we built a system that could fly a drone just from seeing it. Using the Xbox 360 Kinect and a Raspberry Pi, we were able to track a drone in 3D space and control its position based on that information.

We created a Java program to read info from the Kinect and send controls to the Raspberry Pi. We used Python on the Raspberry Pi to control the drone after we wired it into the Drone's remote control. Using this control system we were able to achieve flight for the full duration of the drone's battery life.
